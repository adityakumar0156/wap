Guide
Here you can compose an animated message/letter/note and then generate a link of that content which you can share on Instagram or Facebook.You just need to add elements one by one and click on generate link.

#Buttons & their uses:
Clear : It clears the text you have entered.
GIFs : It shows the GIFs/memes which you can add in your element.
My Page : It shows you content.
Generate Link : It generates a link of that content each time you click on this button.
Preview : It shows your content in full screen.
Copy :Copy the link into your clipboard.
Delete : Deletes the link form the server.

#Options
Shape : You can select shape of the element which you are going to add in the content
Color: You can select background color of the element.
Font Style : Here you can select a particular font style of a particular element.
Font Size: Here you can select the font size of a particular element.
Image Size : Here you can select the size of image(emojis,GIFs) of a particular element.
Music : here you can select the background music of the content which will play when any one opnes the content.
BackGround : It is the background color of the whole content.

#If you want to remove the selected image, you just need to click on that image on the element.
# If you want to delete any element from the content then you just need to click on the element.
